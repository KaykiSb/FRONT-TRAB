LGPD

A Lei Geral de Proteção de Dados estabelece regras para coleta, armazenamento e uso de dados pessoais no Brasil.

O que é a LGPD?

A LGPD (Lei nº 13.709/2018) protege os direitos de privacidade dos cidadãos, garantindo maior controle sobre suas informações pessoais.

🔍 Acesso aos dados

O usuário pode saber quais dados são coletados.

✏ Correção

Permite corrigir informações incorretas.

🗑 Exclusão

Direito de solicitar remoção de dados.

🔐 Consentimento

Empresas precisam autorização para uso de dados.

Importância da LGPD

A LGPD aumenta a transparência e reduz abusos no uso de informações pessoais, principalmente em redes sociais, bancos e plataformas digitais.
